A modern, purely client-side suite of financial micro-tools built to simplify everyday math. This project was developed as a Custom Software Developer trial submission for **Digital Heroes**.

## 🎯 Why I Built This
Standard online calculators often fall short—they either lack Indian currency formatting (₹), force you to calculate single items instead of dynamic lists, or hide their best features behind paywalls. I wanted a fast, ad-free hub that handles the specific financial math I use frequently: splitting complex restaurant bills, projecting loan amortization, and generating quick GST invoice totals.

## ✨ Core Features
* **Smart Bill Splitter:** Calculates pre-tip tax additions automatically and provides a clear breakdown of exactly what constitutes the final per-person charge.
* **Advanced EMI & Loan Analyzer:** Features a visual "Principal vs. Interest" distribution bar and instantly calculates total interest over the loan tenure.
* **Dynamic GST Builder:** Allows users to add or remove multiple line items with varying tax brackets (5%, 12%, 18%, 28%) to build a complete invoice total, rather than just calculating one static number.
* **Dark Mode Ready:** Fully integrated system-preference dark mode using Tailwind CSS.
